# Projeto de Refatoração de Monolítico para Microserviço usando o Padrão StranglerFigApplication

Este projeto foi desenvolvido para para aprender e aplicar a técnica de refatoração de um monolítico para um microserviço. A abordagem utilizada neste projeto é conhecida como Strangler Fig Application, um padrão amplamente utilizado para refatorar sistemas monolíticos de grande escala.

Para saber mais sobre o padrão Strangler Fig Application, você pode consultar o artigo de Martin Fowler: [Strangler Fig Application](https://martinfowler.com/bliki/StranglerFigApplication.html).
